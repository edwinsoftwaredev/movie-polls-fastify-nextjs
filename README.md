## Project Structure

- **📁 API**

  Client request handler

  - TypeScript
  - Fastify
  - Amazon DynamoDB SDK
  - tRPC
  - Upstash SDK
  - Vercel SDK
  - Google Sign-In API
  - Vitest

- **📁 Prisma**

  Database schema definition

- **📁 Redis Server**

  Configuration files for development environment

- **📁 tRPC**

  Typed API endpoints

  - TypeScript
  - tRPC
  - Zod

- **📁 Web Client**

  Web App

  - TypeScript
  - Next.js
  - tRPC
  - TanStack Query
  - Sass
